OPC UA модель
=============

Сервер OPC UA создаёт адресное пространство `Objects/CNC` с узлами состояния,
координат дельты, параметров калибровки и методами управления (Start, Hold,
Reset, Home, SaveParams, LoadParams). Встроенные примеры показывают, как
выполнять чтение/запись узлов и вызывать методы с помощью `open62541`.

Дополнительно публикуется показатель `Objects/CNC/Vision/AlignmentError`,
который поступает из подсистемы машинного зрения. Клиенты могут подписываться
на изменения с частотой 10–50 Гц и использовать значение для автоматической
калибровки или аварийного останова при превышении допустимого порога.
