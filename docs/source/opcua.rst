OPC UA модель
=============

Сервер OPC UA создаёт адресное пространство `Objects/CNC` с узлами состояния,
координат дельты, параметров калибровки и методами управления (Start, Hold,
Reset, Home, SaveParams, LoadParams). Встроенные примеры показывают, как
выполнять чтение/запись узлов и вызывать методы с помощью `open62541`.
